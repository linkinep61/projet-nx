# Android Studio — Instructions Permanentes

## Navigation des menus (CRITIQUE)
- **NE JAMAIS cliquer** sur les éléments du menu bar (Git, Build, etc.)
- Cliquer sur le **hamburger (☰)** pour révéler la barre de menu
- Puis **SURVOLER** (hover / mouse_move) les éléments du menu pour ouvrir les **sous-menus flottants**
- Ensuite cliquer sur l'option désirée dans le sous-menu flottant (ex: "Push...")
- L'utilisateur a insisté plusieurs fois sur ce point : "Il faut survoler, pas cliquer, des menus flottants"

## Git dans Android Studio
- Le proxy sandbox bloque git push (erreur 403) — utiliser l'UI d'Android Studio pour push
- Séquence push : Hamburger ☰ → Hover "Git" → Clic "Push..." dans le sous-menu flottant
